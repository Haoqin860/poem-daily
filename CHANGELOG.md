# 更新日志

## v2.0.0 - 2026-05-13

### 前端重构

- 重构 `index.html` 为全新的「诗词阅读书案」界面，桌面端采用检索、正文、阅读信息三栏布局。
- 移动端调整为诗词正文优先展示，避免检索面板压住首屏阅读体验。
- 重新设计宣纸、朱砂、松绿、淡金组成的视觉系统，弱化旧版重黑背景与强装饰感。
- 为长诗题加入紧凑与密集排版档位，减少超长标题对正文阅读的压迫。

### 交互体验

- 保留并优化每日推荐、分类筛选、全文搜索、随机赏读、收藏、分享、诗词解读等原有功能。
- 新增键盘左右方向键切换诗词，`Escape` 关闭解读弹窗。
- 搜索结果与收藏列表改为安全 DOM 渲染，减少内联字符串拼接带来的转义风险。
- 收藏按钮现在会根据当前诗词状态显示「收藏 / 已收藏」。

### 响应式与质量

- 完成桌面 `1280x720` 与移动 `390x844` 视口验证，无页面水平溢出。
- 浏览器控制台验证为 `0 errors, 0 warnings`。
- 保持纯前端、零构建依赖，仍可通过 `python3 -m http.server 8080` 本地启动。
